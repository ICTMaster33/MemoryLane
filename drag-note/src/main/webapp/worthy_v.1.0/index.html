단지 드래그만으로 텍스트 저장

드래그 노트의 크롬 확장프로그램을 통해 인터넷 서핑 중에 원하는 글감을 드래그하세요. 바로 드래그 노트의 드래그 박스에 저장이 됩니다. 언제든지 꺼내 볼 수 있어요.

빠른 노트 작성 그리고 공유 기능까지

검색을 끝낸 뒤 드래그 박스에서 저장해놓은 텍스트를 클릭해보세요. 에디터에 텍스트가 바로 올라와 빠르게 글을 정리할 수 있습니다. 작성한 노트를 메일로 공유할 수 있고, 당신의 PC로 다운로드도 가능합니다.

누구나 손쉽게, 간결한 디자인

드래그와 클릭으로 간편하게 조작이 가능합니다. 누구나 손쉽게 사용이 가능한 노트 클라우드 서비스입니다. 간결한 디자인으로 두고 두고 꺼내보고 싶은 나만의 노트를 작성해보세요.
